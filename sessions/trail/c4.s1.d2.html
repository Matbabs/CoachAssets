Jour 2 : VMA Courte (30/30)

📊 20 min Échauffement + 2 séries de (10 x 30s Z5 / 30s Z1) Intensité : FC >95% de ta FC max / 100-105% de ta VMA (Allure très rapide)

→ Récup entre les séries : 3 min de trot très lent (Z1)

👉 Comment faire l'exo : La séance culte pour muscler le cœur. Les 30s d'effort doivent être intenses mais régulières. Ne sprinte pas sur les deux premières pour t'écrouler ensuite. La récupération de 30s se fait en trot léger pour ne pas laisser le cœur redescendre trop bas.
