Jour 2 : Récupération Active (Drainage)

45 min en Zone 1 (Moulinage) 📊 Protocole :

→ Durée : 45 minutes sur un terrain parfaitement plat.

→ Intensité : Zone 1 uniquement (récupération totale).

→ Cadence : Haute (>95 RPM). Les jambes doivent tourner très vite mais sans aucune résistance dans les pédales.

→ Équipement : Tenue légère, aérez-vous.

👉 Comment faire l'exo :

- Mental : C'est une séance de "bien-être". Si vous avez un home-trainer, c'est l'occasion idéale de l'utiliser devant un film ou un documentaire sans stress.

- Technique : Concentrez-vous sur la vitesse de rotation des chevilles. Le geste doit être aérien.

- Focus :

- Circulation : Le but est de faire circuler le sang pour évacuer les derniers résidus de fatigue des semaines précédentes.

- Zéro Pression : Si vous sentez une tension dans les cuisses, c'est que vous emmenez trop de braquet. Réduisez la résistance.

- Objectif : Éliminer les tensions musculaires et restaurer la fraîcheur mentale avant le prochain bloc.
