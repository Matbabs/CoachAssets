OPTIMISER SA NUTRITION POUR LE GRAVEL : ÉNERGIE, ENDURANCE ET RÉCUPÉRATION Objectif : Adapter son alimentation aux efforts prolongés (3h+), aux variations d’intensité et aux conditions souvent extrêmes (chaleur, froid, dénivelé).

POURQUOI UNE STRATÉGIE NUTRITIONNELLE SPÉCIFIQUE ?

  • Le gravel combine endurance (base aérobie) et pic d’efforts explosifs (montées raides, relances), sollicitant à la fois les réserves de glycogène et les lipides.
  • Une étude de l’International Journal of Sport Nutrition montre que 30 à 90g de glucides/heure sont nécessaires pour éviter l’hypoglycémie réactive après 2h30 d’effort.
  • La déshydratation réduit les performances de 2% par % de poids corporel perdu (ex : -1kg pour 70kg = -2% de puissance).
  • Les micronutriments (sodium, magnésium, potassium) deviennent critiques pour éviter crampes et fatigue centrale.

LE PLAN NUTRITION : AVANT, PENDANT, APRÈS L’EFFORT

  • J-1 à J-2 : Charge en glycogène :
    • Augmenter les glucides complexes (riz basmati, patate douce, quinoa) à 8-10g/kg de poids (ex : 560-700g pour 70kg).
    • Réduire les fibres et graisses pour limiter le confort digestif (éviter légumes crus, fritures).
    • Hydratation : 500ml d’eau + électrolytes toutes les 2h (avec 300-500mg de sodium/L).
  • 3h avant le départ : Petit-déjeuner optimisé :
    • Glucides lents : Porridge avoine + banane (index glycémique moyen).
    • Protéines légères : Œuf poché ou fromage blanc (15-20g) pour limiter la faim précoce.
    • Graisses saines : 1 c. à café de beurre de cacahuète (pour l’énergie à long terme).
    • Hydratation : 500ml d’eau + caféine modérée (1 expresso) si toléré.
  • Pendant l’effort (par heure) :
    • Glucides :
      • 60-90g/h (mix glucose + fructose pour une absorption optimale).
      • Exemples : 2 gels (25g chacun) + 1 barre énergétique (30g) + boisson iso (20g).
      • Éviter les fibres et protéines pendant l’effort (risque de troubles digestifs).
    • Hydratation :
      • 500-750ml/h (adapter selon température).
      • Boisson contenant 300-500mg de sodium/L (ex : Tailwind, High5 Zero).
      • En chaleur (>25°C) : Ajouter 1 cachet d’électrolytes toutes les 30 min.
    • Stratégie pratique :
      • Manger toutes les 30-45 min (réveil l’horloge avant la faim).
      • Privilégier les aliments solides en début d’effort (barres, fruits secs) et les gels/liquides en fin de sortie (fatigue digestive).
      • Tester sa stratégie à l’entraînement avec les mêmes produits qu’en course !
  • Dans les 30 min post-effort : Fenêtre de récupération :
    • Glucides rapides : 1g/kg de poids (ex : 70g pour 70kg) → compote, pain blanc, boisson de récup.
    • Protéines : 20-30g (lait chocolaté, recovery shake, ou œufs brouillés).
    • Réhydratation : 1,5L pour 1kg perdu (vérifier avec une balance avant/après).

ALIMENTS CLÉS POUR LE GRAVEL

  • Avant l’effort :
    • Riz blanc + sauce soja : Glucides faciles à digérer + sodium.
    • Pain d’épices : Index glycémique modéré + anti-inflammatoire (gingembre).
    • Smoothie banane-avoine-lait : Énergie progressive + potassium.
  • Pendant l’effort :
    • Dattes fourrées à la pâte d’amande : Glucides + magnésium (anti-crampes).
    • Barres maison (flocons d’avoine + miel + noix) : Énergie durable et naturelle.
    • Pommes de terre cuites froides : Alternative salée aux gels (riches en potassium).
  • Après l’effort :
    • Omelette aux épinards + patate douce : Protéines + glucides + fer.
    • Yaourt grec + miel + noix : Rééquilibrage glycémique + protéines.
    • Soupe miso + tofu : Sodium + protéines végétales (idéal si appétit coupé).

ERREURS À ÉVITER ET ASTUCES TERRAIN

  • Erreurs courantes :
    • Négliger l’hydratation dès le départ (la soif est un signal tardif).
    • Consommer trop de fibres avant l’effort (ex : salade de légumes crus → ballonnements).
    • Oublier le sel en chaleur (risque d’hyponatrémie).
    • Tester un nouveau gel ou barre le jour de la course (risque de troubles digestifs).
  • Astuces pro :
    • Préparer ses portions de glucides dans des sachets zip (ex : 30g de pâte de dattes = 1 portion).
    • Utiliser une gourde à paille pour boire sans lâcher le guidon.
    • En cas de faim soudaine : Croquer un morceau de pain sec (glucides rapides et rassasiant).
    • Pour les sorties >6h : Ajouter des acides aminés (BCAA) en fin d’effort pour limiter la fatigue centrale.
  • Signaux d’alerte :
    • Crampes : Manque de sodium/magnésium → boisson électrolytique + banane.
    • Nausées : Trop de glucides concentrés → diluer les gels dans l’eau.
    • Coups de barre : Hypoglycémie → gel + barre énergétique immédiatement.
    • Urines foncées : Déshydratation → boire par petites gorgées régulières.

EXEMPLE DE PLAN POUR UNE SORTIE DE 5H

  • 6h00 : Petit-déjeuner
    • 150g de riz blanc + 1 œuf + 1 banane + 500ml eau électrolytique.
  • 7h00 : Départ
    • Emporter : 5 gels (25g chacun), 2 barres, 2L de boisson iso + 1L d’eau pure.
  • Pendant l’effort
    • Heure 1 : 1 gel + 500ml boisson.
    • Heure 2 : 1 barre + 500ml eau.
    • Heure 3 : 1 gel + 1/2 banane séchée + 500ml boisson.
    • Heure 4 : 1 gel + 2 dattes + 500ml eau.
    • Heure 5 : 1 gel + 1/2 barre + 500ml boisson.
  • Fin d’effort : Récupération
    • Dans les 30 min : Shaker récup (30g protéines + 50g glucides) + 500ml eau.
    • Repas dans les 2h : Pâtes + poulet + légumes vapeur + compote.
